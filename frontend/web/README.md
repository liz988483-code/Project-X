# SOKO Marketplace - Frontend

A modern e-commerce marketplace built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 🛍️ Multi-vendor marketplace
- 👤 User authentication (Buyer/Seller)
- 🛒 Shopping cart with Zustand
- 💳 Checkout process
- 📱 Responsive design
- 🎨 Modern UI with Tailwind CSS
- 🔐 Secure authentication
- 📊 Product search and filtering
- ⭐ Product reviews and ratings
- 📦 Order management
- 💬 Real-time chat (coming soon)
- 📈 Seller dashboard
